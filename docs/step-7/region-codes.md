# 행정구역 코드 목록 (LibraryInfoNaru API)

> **버전:** v20250306 (전면 개정)
> **출처:** 도서관정보나루 API (https://www.data4library.kr)

## 📌 개요

이 문서는 도서관정보나루 API에서 사용하는 **행정구역 코드**를 정리한 것입니다.

### 코드 구조
```
[시/도 코드 2자리] + [시/군/구 코드 3자리] = 5자리 전체 코드
```

**예시:**
- `11010` = `11`(서울) + `010`(종로구)
- `24030` = `24`(광주) + `030`(남구)

## 📝 참고사항

1. **코드 체계**
   - 2자리: 시/도 단위 (예: 11=서울, 24=광주)
   - 5자리: 시/군/구 단위 (예: 11010=서울 종로구)

2. **사용 예시**
   ```java
   // 시/도 단위 조회
   region=24  // 광주광역시 전체

   // 세부 지역 조회
   region=24030  // 광주광역시 남구
   ```

3. **API 적용**
   - 도서관정보나루 API의 `region` 파라미터에 사용
   - 일부 API는 2자리 코드만 지원
   - 상세 조회는 5자리 코드 사용 가능
