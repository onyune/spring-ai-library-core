## 1. Library Assistant Agent란?

### 정의

> **Library Assistant Agent** = 사용자의 자연어 질문을 이해하고, 필요한 도구(함수)를 자동으로 선택하여 도서관 서비스를 제공하는 **지능형 도서관 도우미 AI 시스템**

### 핵심 기능

1. **도서 검색**: 자연어로 도서 검색 (제목, 저자, 키워드)
2. **리뷰 요약**: 도서별 리뷰 요약 및 평점 제공
3. **대출 가능 여부**: 전국 도서관 대출 가능 현황 실시간 조회
4. **개인화 추천**: 사용자 선호도(피드백) 반영한 맞춤 추천

### 기존 방식과의 차이

#### ❌ 기존 방식 (Step 5 Telegram Bot)

사용자가 각 기능을 별도로 호출해야 함:

```
사용자: "자바의 정석" 검색
Bot: [검색 결과 3권 표시]

사용자: "1번 리뷰 보여줘"
Bot: [별도 API 호출 → 리뷰 표시]

사용자: "대출 가능해?"
Bot: [또 별도 조회 필요]
```

**문제점**:
- 사용자가 여러 번의 질문 필요
- 각 기능을 따로 호출해야 함
- 정보가 분산되어 있음

#### ✅ Library Assistant Agent 방식

사용자의 한 번의 질문으로 모든 정보를 제공:

```
사용자: "자바의 정석 책 정보, 리뷰, 대출 가능한 곳 알려줘"
   ↓
[AI 분석] → 필요한 도구(함수) 식별
   ↓
[1단계] 도서 검색 실행
[2단계] 리뷰 조회 실행
[3단계] 대출 가능 도서관 조회
   ↓
[AI 종합] "자바의 정석을 찾았습니다!
  - 평점: 4.7/5.0 (328개 리뷰)
  - 리뷰: '설명이 체계적이고 예제가 풍부합니다'
  - 대출 가능: 강남도서관(1.2km), 역삼도서관(2.1km)"
```

## 2. 왜 Library Assistant Agent인가?

### 2.1. 사용자 경험 개선

**문제**: 사용자는 도서 정보를 얻기 위해 여러 단계를 거쳐야 함

**해결**: 한 번의 질문으로 모든 정보 제공
- 시간 절약: 3번의 질문 → 1번의 질문
- 정보 통합: 검색, 리뷰, 대출 정보를 한 곳에서
- 자연스러운 대화: 자연어 질문으로 원하는 정보 얻기

### 2.2. Step 6 기반과의 완벽한 시너지

이미 구현된 기능들을 조합하기만 하면 됩니다:

| Step 6 기능 | Library Assistant Agent에서의 활용 |
|-------------|-------------------------------------|
| **BookSearchService** | 도서 검색 Function |
| **ReviewService** | 리뷰 조회 Function |
| **PersonalizationService** | 개인화 Function |
| **LibraryInfoNaruApiClient** | 대출 가능 여부 Function |

**장점**:
- 새로운 기능 구현 없이 기존 코드 재활용
- 이미 검증된 서비스를 활용하여 안정성 확보
- 개발 시간 단축

### 2.3. 교육적 가치

**학생들이 습득할 수 있는 기술**:

1. **Spring AI Function Calling**
   - AI가 시스템 도구를 사용하는 방법
   - @Description 어노테이션 활용
   - ChatClient 설정

2. **오케스트레이션 패턴**
   - 여러 함수의 조합
   - 함수 간 데이터 전달
   - 순차적 실행 흐름 제어

3. **외부 API 연동**
   - 도서관정보나루 API 연동
   - Fallback 처리
   - 에러 복구

4. **실무 경험**
   - AI 시스템 설계
   - 자동화된 워크플로우 구축
   - 포트폴리오 작품

## 3. 주요 시나리오

### 시나리오 1: 기본 도서 검색

```
사용자: "자바 책 5권 추천해줘"

Library Assistant Agent 동작:
1. 📖 검색 Function 실행
   → BookSearchService.searchBooks("자바", HYBRID)

2. 👤 개인화 적용
   → PersonalizationService.personalizedSearch(results, chatId)

3. 💬 AI 답변 생성
   "자바 관련 도서 5권을 추천합니다:
   1. 자바의 정석 (평점 4.7)
   2. 이것이 자바다 (평점 4.5)
   ..."
```

### 시나리오 2: 도서 상세 정보 통합 조회

```
사용자: "자바의 정석 책 정보랑 리뷰도 알려줘"

Library Assistant Agent 동작:
1. 📖 도서 검색 Function
   → BookSearchService.searchBooks("자바의 정석")

2. 📊 리뷰 조회 Function
   → ReviewService.getReviewSummary(bookId)

3. 💬 종합 답변
   "자바의 정석을 찾았습니다!
   - 저자: 남궁성
   - 출판사: 도우출판
   - 평점: 4.7/5.0 (328개 리뷰)
   - 리뷰 요약: '설명이 체계적이고 예제가 풍부합니다'
   - 좋은 점: '개념 설명이 명확함'
   - 나쁜 점: '분량이 많아 입문자에게 부담될 수 있음'"
```

### 시나리오 3: 대출 가능 여부 확인 (핵심 시나리오)

```
사용자: "내 주변에서 빌릴 수 있는 '자바의 정석' 책 알려줘"

Library Assistant Agent 동작:
1. 📖 도서 검색 Function
   → BookSearchService.searchBooks("자바의 정석")
   → ISBN: 9788960777331 확보

2. 📚 대출 가능 여부 Function
   → LibraryInfoNaruApiClient.getLoanAvailability("9788960777331")

3. 💬 종합 답변
   "자바의 정석을 찾았습니다!
   - ISBN: 9788960777331
   - 대출 가능 도서관 (총 15개소):
     1. 강남도서관 (거리 1.2km) - 대출 가능 ✅
     2. 역삼도서관 (거리 2.1km) - 대출 가능 ✅
     3. 서초도서관 (거리 3.5km) - 대출 가능 ✅
     ...
   - 가장 가까운 곳: 강남도서관 (1.2km)
   - 대출 전 화면 예약: 강남도서관 홈페이지에서 가능"
```

```
┌─────────────────────────────────────────────────────────────────┐

│  │ Function      │    │  Function     │    │  Function     │   │
│  └───────┬───────┘    └───────┬───────┘    └───────┬───────┘   │
│          │                    │                    │             │
│          └────────────────────┼────────────────────┘             │
│                               │                                  │
│                    ┌───────────▼────────────┐                    │
│                    │  ChatClient (Spring AI)│                    │
│                    │   + Orchestrator       │                    │
│                    └───────────┬────────────┘                    │
│                                │                                  │
│  ┌───────────────────────┼───────────────────────┐             │
│  │                       │                       │             │
│  ▼                      ▼                      ▼             │
│  사용자 질문    →   [AI 분석]   →   함수 자동 선택    │
│  (자연어)              (의도 파악)   (Function Calling)  │
│                                  │                   │
│                                  ▼                   │
│                         [Functions 실행]                    │
│                                  │                   │

└─────────────────────────────────────────────────────────────┘
```

### 4.2. 데이터 흐름도

```
[Step A] 사용자 입력
    ↓
"자바의 정석 책 정보, 리뷰, 대출 가능한 곳 알려줘"
    ↓
[Step B] AI 분석 (Spring AI ChatClient)
    ├─ 의도 파악: 도서 검색 + 리뷰 + 대출 확인
    ├─ ISBN 필요: 검색 먼저 실행
    └─ chatId 필요: 개인화에 사용
    ↓
[Step C] Function 자동 선택 및 실행
    ├─ 1. searchBooks("자바의 정석")
    │   → BookSearchService
    │   → 결과: { bookId: 123, isbn: "9788960777331" }
    │
    ├─ 2. getReviews(123)
    │   → ReviewService
    │   → 결과: { summary: "...", rating: 4.7 }
    │
    ├─ 3. checkLoanAvailability("9788960777331")
    │   → LibraryInfoNaruApiClient
    │   → 결과: [{ library: "강남", distance: 1.2km }, ...]
    │
    └─ 4. personalizedSearch(results, chatId)
    │   → PersonalizationService
    │   → 결과: 사용자 선호도 반영한 재정렬
    ↓
[Step D] AI 종합 답변 생성
"자바의 정석을 찾았습니다! (상세 정보 + 리뷰 + 대출 가능 도서관)"
```

## 5. 구현 난이도와 시간

### 단계별 난이도

| 단계 | 내용 | 난이도 | 예상 시간 |
|------|------|--------|----------|
| **이론** | Function Calling 개념 | ⭐☆☆☆☆ | 1시간 |
| **Level 1** | 단일 Function (도서 검색) | ⭐★☆☆☆ | 2시간 |
| **Level 2** | 복수 Function (검색 + 리뷰) | ⭐⭐⭐☆☆ | 3시간 |
| **Level 3** | 외부 API 연동 (대출 가능) | ⭐⭐⭐⭐☆ | 2시간 |
| **테스트** | 통합 테스트 및 디버깅 | ⭐⭐⭐☆☆ | 1시간 |
| **총계** | | **⭐⭐☆☆ (중급)** | **9시간** |

### 초급자에게 도움이 되는 요소

1. **기존 코드 재활용**
   - 이미 익숙한 BookSearchService 등 활용
   - 새로운 패턴이 아니라 기존 것의 확장

2. **점진적 학습**
   - 단일 Function → 복수 Function → 외부 API
   - 각 단계별로 명확한 목표

3. **실습 중심**
   - 이론 최소화, 실습 최대화
   - 코드 복사 가능한 예시 제공

## 6. 기술 스택 상세

### 6.1. Spring AI Function Calling

**핵심 컴포넌트**:
- **ChatClient**: AI와 함수를 연결하는 인터페이스
- **@Description**: AI에게 함수 용도를 설명하는 어노테이션
- **Function<Request, Response>**: 함수형 인터페이스

### 6.2. 활용 기술 (Step 6 이미 구현됨)

| 컴포넌트 | 패키지 경로 | 용도 |
|---------|-----------|------|
| BookSearchService | `core.book.service.search` | 도서 검색 |
| ReviewService | `core.review.service` | 리뷰 관리 |
| PersonalizationService | `core.book.service.personalization` | 개인화 |
| LibraryInfoNaruApiClient | `external.opennaru.client` | 도서관정보나루 API |

### 6.3. 새로 추가될 컴포넌트

| 컴포넌트 | 패키지 경로 | 용도 |
|---------|-----------|------|
| LibraryAssistantService | `ai.service` | 메인 서비스 |
| BookSearchFunction | `ai.function` | 검색 Function |
| ReviewFunction | `ai.function` | 리뷰 Function |
| LoanCheckFunction | `ai.function` | 대출 확인 Function |
| AiConfig | `core.config` | ChatClient 설정 |

## 7. 예상 결과

### 구현 완료 후 사용자 경험

**입력**:
```
사용자: "내 주변에서 빌릴 수 있는 '이것이 자바다' 책 알려줘"
```

**출력**:
```
Library Assistant:
"이것이 자바다를 찾았습니다! 📚

📖 도서 정보:
- 제목: 이것이 자바다
- 저자: 박매일
- 출판사: 한빛미디어
- ISBN: 9788960777330

⭐ 평점: 4.6/5.0 (247개 리뷰)

📝 리뷰 요약:
'개념을 코드로 구현하는 과정을 매우 상세히 설명합니다.

🏛️ 대출 가능 도서관 (총 8개소):
1. 강남도서관 (거리 0.8km) ✅ 대출 가능
   - 주소: 서울 강남구 테헤란로 123
   - 전화: 02-1234-5678
   - 운영시간: 평일 09:00-18:00

2. 역삼도서관 (거리 1.5km) ✅ 대출 가능
   - 주소: 서울 역삼동 테헤란로 456
   - 전화: 02-2345-6789

3. 서초도서관 (거리 2.8km) ✅ 대출 가능
   ...

💡 추천 이유:
귀하의 '웹 개발' 관심사를 반영하여,
프로젝트 중심의 실무 자바 도서를 함께 추천합니다:

- 스프링 부트实战 (평점 4.5)
- 클린 코드 (평점 4.4)

가장 가까운 곳: 강남도서관 (0.8km)
도서관 방문 전 홈페이지에서 대출 예약 가능합니다! 🏃‍♂️"
```
