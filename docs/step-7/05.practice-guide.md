이 문서는 **단계별 실습 미션**과 **테스트 시나리오**를 통해 Library Assistant Agent를 직접 구현해 볼 수 있는 가이드입니다.

> **🔧 리팩토링 업데이트**: 이 실습 가이드는 최신 리팩토링 내용을 반영하여 업데이트되었습니다.
> - FunctionToolCallback.builder() 방식으로 Function 등록
> - ChatClient를 Bean으로 등록하여 재사용

### 미션 2-3: BookSearchFunction 구현

**파일**: `ai/function/BookSearchFunction.java`

```java
package com.nhnacademy.library.ai.function;

import com.nhnacademy.library.ai.function.dto.BookSearchRequest;
import com.nhnacademy.library.ai.function.dto.BookSearchResult;
import com.nhnacademy.library.core.book.dto.BookSearchResponse;
import com.nhnacademy.library.core.book.service.search.BookSearchService;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.context.annotation.Description;
import org.springframework.stereotype.Component;

import java.util.function.Function;
import java.util.stream.Collectors;

/**
 * 도서 검색 Function
 */
@Component
@RequiredArgsConstructor
@Slf4j
public class BookSearchFunction {

    private final BookSearchService bookSearchService;

    @Description("도서관 시스템에서 도서를 검색합니다. " +
            "제목, 저자명으로 검색할 수 있습니다.")
    public Function<BookSearchRequest, BookSearchResult> searchBooks() {
        return request -> {
            log.info("[Function] 도서 검색 시작: query={}", request.query());

            try {
                // 1. 내부 DB 검색 실행
                var books = bookSearchService.searchBooks(
                    com.nhnacademy.library.core.book.dto.BookSearchRequest.builder()
                        .query(request.query())
                        .searchType(com.nhnacademy.library.core.book.domain.SearchType.HYBRID)
                        .build()
                );

                // 2. 결과 변환
                var items = books.getContent().stream()
                    .map(this::toBookItem)
                    .collect(Collectors.toList());

                log.info("[Function] 검색 완료: {}권", items.size());

                // 3. 결과 반환
                return BookSearchResult.builder()
                    .count(items.size())
                    .books(items)
                    .build();

            } catch (Exception e) {
                log.error("[Function] 검색 실패", e);
                return BookSearchResult.builder()
                    .count(0)
                    .books(java.util.List.of())
                    .build();
            }
        };
    }

    private BookSearchResult.BookItem toBookItem(BookSearchResponse book) {
        return BookSearchResult.BookItem.builder()
            .id(book.id())
            .title(book.title())
            .author(book.authorName())
            .publisher(book.publisherName())
            .build();
    }
}
```

### 미션 2-4: AiConfig 수정

**파일**: `core/config/AiConfig.java`

> **중요**: 리팩토링 후에는 AiConfig에서 **FunctionToolCallback을 Bean으로 등록**하고, ChatClient도 Bean으로 등록합니다.

```java
package com.nhnacademy.library.core.config;

import com.nhnacademy.library.ai.function.BookSearchFunction;
import com.nhnacademy.library.ai.function.dto.BookSearchRequest;
import com.nhnacademy.library.ai.function.dto.BookSearchResult;
import lombok.RequiredArgsConstructor;
import org.springframework.ai.chat.client.ChatClient;
import org.springframework.ai.chat.model.ChatModel;
import org.springframework.ai.tool.function.FunctionToolCallback;
import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.boot.autoconfigure.condition.ConditionalOnProperty;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
@RequiredArgsConstructor
public class AiConfig {

    private final ChatModel chatModel;
    private final BookSearchFunction bookSearchFunction;

    /**
     * 도서 검색 FunctionToolCallback Bean
     * <p>성능 최적화를 위해 Bean으로 등록하여 재사용합니다.</p>
     */
    @Bean
    public FunctionToolCallback<BookSearchRequest, BookSearchResult> bookSearchTool(
        BookSearchFunction bookSearchFunction) {
        return FunctionToolCallback.builder("BookSearch", bookSearchFunction::searchBooks)
            .description("도서관 시스템에서 도서를 검색합니다. " +
                        "제목, 저자명으로 검색할 수 있으며, 검색어를 받아 도서 목록을 반환합니다.")
            .inputType(BookSearchRequest.class)
            .build();
    }

    /**
     * AiLibraryAssistantService용 ChatClient Bean
     * <p>FunctionToolCallback이 미리 등록된 ChatClient입니다.</p>
     */
    @Bean
    @ConditionalOnProperty(name = "spring.ai.selected-model", havingValue = "ollama")
    public ChatClient libraryAssistantChatClient(
        @Qualifier("ollamaChatModel") ChatModel chatModel,
        FunctionToolCallback<BookSearchRequest, BookSearchResult> bookSearchTool) {
        return ChatClient.builder(chatModel)
            .defaultToolCallbacks(bookSearchTool)
            .build();
    }

    /**
     * Gemini용 ChatClient Bean
     */
    @Bean
    @ConditionalOnProperty(name = "spring.ai.selected-model", havingValue = "gemini", matchIfMissing = true)
    public ChatClient libraryAssistantChatClientGemini(
        @Qualifier("googleGenAiChatModel") ChatModel chatModel,
        FunctionToolCallback<BookSearchRequest, BookSearchResult> bookSearchTool) {
        return ChatClient.builder(chatModel)
            .defaultToolCallbacks(bookSearchTool)
            .build();
    }
}
```

**설명**:
1. `bookSearchTool()`: FunctionToolCallback을 Bean으로 등록하여 재사용
2. `libraryAssistantChatClient()`: FunctionToolCallback이 포함된 ChatClient를 Bean으로 등록
3. `@Qualifier`: 사용할 ChatModel을 명시적으로 지정

### 미션 2-5: AiLibraryAssistantService 구현

**파일**: `ai/service/AiLibraryAssistantService.java`

> **중요**: 리팩토링 후에는 ChatClient를 **주입받아 사용**합니다.

```java
package com.nhnacademy.library.ai.service;

import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.ai.chat.client.ChatClient;
import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.stereotype.Service;

/**
 * Library Assistant Agent 메인 서비스
 * <p>
 * <b>리팩토링:</b> ChatClient를 Bean으로 주입받아 재사용합니다.
 * </p>
 */
@Service
@RequiredArgsConstructor
@Slf4j
public class AiLibraryAssistantService {

    @Qualifier("libraryAssistantChatClient")
    private final ChatClient chatClient; // 주입받아 재사용

    /**
     * 사용자 요청을 처리하고 AI 응답을 반환합니다.
     *
     * @param userMessage 사용자 메시지
     * @return AI 응답
     */
    public String ask(String userMessage) {
        log.info("[AI Service] 사용자 요청: {}", userMessage);

        // System Prompt
        String systemPrompt = """
            당신은 도서관 도우미 AI입니다.

            **중요: 반드시 한국어로만 답변하세요.**

            사용자의 요청을 처리할 때:
            1. 필요한 도구(함수)를 호출하여 정보를 수집하세요
            2. 도구 호출 결과를 받으면 반드시 자연스러운 한국어로 답변하세요
            3. 검색 결과를 친절하게 설명하세요
            """;

        String response = chatClient.prompt()
            .system(systemPrompt)
            .user(userMessage)
            .call()
            .content();

        log.info("[AI Service] AI 응답 생성 완료");
        return response;
    }
}
```

**설명**:
1. `@Qualifier("libraryAssistantChatClient")`: AiConfig에서 등록한 ChatClient Bean을 주입
2. `systemPrompt`: AI가 한국어로 답변하도록 지시
3. ChatClient를 매번 생성하지 않고 재사용하여 성능 향상

### 미션 3-2: ReviewFunction 구현

**파일**: `ai/function/ReviewFunction.java`

> **중요**: 리팩토링 후에는 **@Description 어노테이션 대신 직접 메서드로 구현**합니다.

```java
package com.nhnacademy.library.ai.function;

import com.nhnacademy.library.ai.function.dto.ReviewResult;
import com.nhnacademy.library.core.review.dto.ReviewResponse;
import com.nhnacademy.library.core.review.service.ReviewService;
import com.nhnacademy.library.core.book.service.search.BookSearchService;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.data.domain.PageRequest;
import org.springframework.stereotype.Component;

import java.util.List;

/**
 * 리뷰 조회 Function (Spring AI 1.1.2 FunctionToolCallback 방식)
 * <p>
 * AI Agent가 도서의 리뷰를 조회할 때 호출하는 함수입니다.
 * 도서 ID를 받아 리뷰 요약, 평점, 주요 리뷰를 반환합니다.
 * </p>
 */
@Component
@RequiredArgsConstructor
@Slf4j
public class ReviewFunction {

    private final ReviewService reviewService;
    private final BookSearchService bookSearchService;

    private static final int REVIEW_LIMIT = 5;

    /**
     * 리뷰 조회 실행 메서드 (외부에서 호출)
     * <p>
     * FunctionToolCallback.builder()에서 메서드 레퍼런스로 사용합니다.
     * </p>
     */
    public ReviewResult getReviews(Long bookId) {
        log.info("[AI Function] 리뷰 조회 시작: bookId={}", bookId);

        try {
            // 1. 리뷰 조회 (상위 N개)
            var reviews = reviewService.getReviews(
                bookId,
                PageRequest.of(0, REVIEW_LIMIT)
            );

            // 2. 리뷰 요약 조회
            String summary = bookSearchService.getReviewSummary(bookId);

            // 3. 평균 평점 계산
            double avgRating = reviews.getContent().stream()
                .mapToInt(ReviewResponse::rating)
                .average()
                .orElse(0.0);

            // 4. 결과 변환
            List<ReviewResult.ReviewItem> reviewItems = reviews.getContent().stream()
                .map(review -> ReviewResult.ReviewItem.of(
                    review.content(),
                    review.rating()
                ))
                .toList();

            log.info("[AI Function] 리뷰 조회 완료: bookId={}, reviewCount={}",
                bookId, reviewItems.size());

            // 5. 결과 반환
            return ReviewResult.of(
                bookId,
                summary,
                avgRating,
                (int) reviews.getTotalElements(),
                reviewItems
            );

        } catch (Exception e) {
            log.error("[AI Function] 리뷰 조회 실패: bookId={}", bookId, e);
            return ReviewResult.empty(bookId);
        }
    }
}
```

**설명**:
1. `@Description` 어노테이션 사용하지 않음
2. 직접 `getReviews()` 메서드로 구현
3. `ReviewResult.of()`, `empty()` 정적 팩토리 메서드 사용

                double avgRating = reviews.getContent().stream()

                        ))

                    bookId, reviewItems.size());

                // 5. 결과 반환
                return ai.function.dto.ReviewResult.builder()
                    .bookId(bookId)
                    .summary(summary)
                    .rating(avgRating)
                    .reviewCount((int) reviews.getTotalElements())

### 미션 4-3: LoanCheckFunction 구현

**파일**: `ai/function/LoanCheckFunction.java`

> **중요**: 리팩토링 후에는 **@Description 어노테이션 대신 직접 메서드로 구현**합니다.

```java
package com.nhnacademy.library.ai.function;

import com.nhnacademy.library.ai.function.dto.LoanCheckRequest;
import com.nhnacademy.library.ai.function.dto.LoanCheckResult;
import com.nhnacademy.library.external.opennaru.client.LibraryInfoNaruApiClient;
import com.nhnacademy.library.external.opennaru.dto.LoanItemInfo;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.util.List;
import java.util.stream.Collectors;

/**
 * 대출 가능 여부 조회 Function (Spring AI 1.1.2 FunctionToolCallback 방식)
 * <p>
 * AI Agent가 도서의 대출 가능 여부를 조회할 때 호출하는 함수입니다.
 * ISBN을 받아 전국 도서관의 대출 가능 현황을 반환합니다.
 * </p>
 */
@Component
@RequiredArgsConstructor
@Slf4j
public class LoanCheckFunction {

    private final LibraryInfoNaruApiClient libraryInfoNaruApiClient;

    /**
     * 대출 가능 확인 실행 메서드 (외부에서 호출)
     * <p>
     * FunctionToolCallback.builder()에서 메서드 레퍼런스로 사용합니다.
     * </p>
     */
    public LoanCheckResult checkLoanAvailability(LoanCheckRequest request) {
        log.info("[AI Function] 대출 가능 여부 조회 시작: isbn={}", request.isbn());

        try {
            // 1. 도서관정보나루 API 호출
            List<LoanItemInfo> loanItems = libraryInfoNaruApiClient.searchLoanItemsByIsbn(
                request.isbn()
            );

            if (loanItems.isEmpty()) {
                log.info("[AI Function] 대출 가능 도서관 없음: isbn={}", request.isbn());
                return LoanCheckResult.empty(request.isbn());
            }

            // 2. 지역 필터링 (선택 사항)
            List<LoanItemInfo> filteredItems = filterByRegion(loanItems, request.region());

            // 3. 상위 N개 추출
            List<LoanItemInfo> topItems = filteredItems.stream()
                .limit(request.limit())
                .toList();

            // 4. 결과 변환
            List<LoanCheckResult.LibraryInfo> libraries = topItems.stream()
                .map(this::toLibraryInfo)
                .toList();

            log.info("[AI Function] 대출 가능 도서관 조회 완료: isbn={}, count={}",
                request.isbn(), libraries.size());

            return LoanCheckResult.of(
                request.isbn(),
                loanItems.size(),
                libraries
            );

        } catch (Exception e) {
            log.error("[AI Function] 대출 가능 여부 조회 실패: isbn={}", request.isbn(), e);
            return LoanCheckResult.empty(request.isbn());
        }
    }

    /**
     * 지역 기반 필터링
     */
    private List<LoanItemInfo> filterByRegion(List<LoanItemInfo> loanItems, String region) {
        if (region == null || region.isBlank()) {
            return loanItems;
        }
        return loanItems.stream()
            .filter(item -> item.location() != null && item.location().contains(region))
            .collect(Collectors.toList());
    }

    /**
     * LoanItemInfo를 LibraryInfo로 변환
     */
    private LoanCheckResult.LibraryInfo toLibraryInfo(LoanItemInfo item) {
        boolean loanAvailable = "Y".equalsIgnoreCase(item.loanAvailable());

        return LoanCheckResult.LibraryInfo.of(
            item.libraryName(),
            item.location(),
            null, // 거리 정보 (현재 미지원)
            "",   // 전화번호 (현재 미지원)
            loanAvailable,
            ""    // 운영 시간 (현재 미지원)
        );
    }
}
```

**설명**:
1. `@Description` 어노테이션 사용하지 않음
2. 직접 `checkLoanAvailability()` 메서드로 구현
3. `LoanCheckResult.of()`, `empty()` 정적 팩토리 메서드 사용
4. 지역 필터링 로직 포함

            try {
                // 1. API 호출
                var availability = libraryInfoNaruApiClient.getLoanAvailability(
                    request.isbn()
                );

                // 2. 결과 변환
                var libraries = availability.libraries().stream()
                    .limit(request.limit())
                    .map(lib -> LoanCheckResult.LibraryInfo.builder()
                        .name(lib.name())
                        .address(lib.address())
                        .distance(lib.distance())
                        .phone(lib.phone())
                        .loanAvailable(lib.loanAvailable())
                        .operatingHours(lib.operatingHours())
                        .build())
                    .toList();

                log.info("[Function] 대출 가능 도서관 조회 완료: count={}", libraries.size());

                return LoanCheckResult.builder()
                    .isbn(request.isbn())
                    .totalLibraries((int) availability.libraries().size())
                    .libraries(libraries)
                    .build();

            } catch (Exception e) {
                log.error("[Function] 대출 가능 여부 조회 실패", e);
                return LoanCheckResult.builder()
                    .isbn(request.isbn())
                    .totalLibraries(0)
                    .libraries(java.util.List.of())
                    .build();
            }
        };
    }
}

**문제 3**: 함수 호출 순서가 잘못됨

**원인**: AI가 필요한 데이터를 확보하지 못함

**해결**: AiLibraryAssistantService의 systemPrompt에 순서 명시
```java
// AiLibraryAssistantService.java
public String ask(String userMessage) {
    String systemPrompt = """
        당신은 도서관 도우미 AI입니다.

        **중요: 반드시 한국어로만 답변하세요.**

        복잡한 요청 처리 순서:
        1. 도서 검색이 필요하면 BookSearch 호출
        2. 리뷰가 필요하면 GetReviews 호출 (bookId 필요)
        3. 대출 가능 여부가 필요하면 CheckLoanAvailability 호출 (ISBN 필요)

        사용자에게 친절하고 상세한 답변을 제공하세요.
        """;

    return chatClient.prompt()
        .system(systemPrompt)
        .user(userMessage)
        .call()
        .content();
}
```
