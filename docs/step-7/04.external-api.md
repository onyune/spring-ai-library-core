## 2. 외부 API 연동이란?

### 2.1. 내부 DB vs 외부 API

**내부 DB 검색** (기존):
```
사용자: "자바 책 찾아줘"
   ↓
[내부 DB 조회]
   ↓
"3권을 찾았습니다..."
```

**외부 API 연동** (이 문서):
```
사용자: "내 주변에서 빌릴 수 있는 자바 책 알려줘"
   ↓
[1단계] 내부 DB 검색 → ISBN 확보
   ↓
[2단계] 도서관정보나루 API 호출 → 대출 가능 도서관 조회
   ↓
[3단계] 거리 기반 필터링 → 가까운 도서관 우선
   ↓
"강남도서관(1.2km)에서 대출 가능합니다!"
```

### 2.2. API 연동 시 고려사항

| 항목 | 고려사항 | 해결 방법 |
|-----|---------|----------|
| **응답 시간** | 외부 API는 느릴 수 있음 | 타임아웃 설정, 비동기 처리 |
| **API 장애** | 외부 서비스 다운 시 대응 | Fallback to 캐시 또는 에러 메시지 |
| **중복 제거** | 같은 도서관이 여러 번 나올 수 있음 | 도서관 ID로 그룹화 |
| **데이터 양** | 전국 도서관 데이터가 너무 많음 | 상위 N개만 반환, 거리 필터링 |

## 3. LoanCheckFunction 구현

### 3.1. LoanCheckRequest DTO

**LoanCheckRequest.java** (`ai/function/dto/`):

```java
package com.nhnacademy.library.ai.function.dto;

import lombok.Builder;

/**
 * 대출 가능 여부 조회 요청
 *
 * @param isbn ISBN (필수)
 * @param region 지역 코드 (선택, 예: "서울", "경기")
 * @param latitude 위도 (선택, 거리 계산용)
 * @param longitude 경도 (선택, 거리 계산용)
 * @param limit 최대 결과 수 (기본값 10)
 */
@Builder
public record LoanCheckRequest(
    String isbn,
    String region,
    Double latitude,
    Double longitude,
    int limit
) {
    public static LoanCheckRequest of(String isbn) {
        return new LoanCheckRequest(isbn, null, null, null, 10);
    }

    public static LoanCheckRequest of(String isbn, String region) {
        return new LoanCheckRequest(isbn, region, null, null, 10);
    }
}
```

package com.nhnacademy.library.ai.function.dto;

import lombok.Builder;

### 3.3. LoanCheckFunction 구현

**LoanCheckFunction.java** (`ai/function/`):

```java
package com.nhnacademy.library.ai.function;

import com.nhnacademy.library.ai.function.dto.LoanCheckRequest;
import com.nhnacademy.library.ai.function.dto.LoanCheckResult;
import com.nhnacademy.library.external.opennaru.client.LibraryInfoNaruApiClient;
import com.nhnacademy.library.external.opennaru.dto.LoanItemInfo;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.stereotype.Component;

import java.util.List;
import java.util.stream.Collectors;

/**
 * 대출 가능 여부 조회 Function (Spring AI 1.1.2 FunctionToolCallback 방식)
 * <p>
 * AI Agent가 도서의 대출 가능 여부를 조회할 때 호출하는 함수입니다.
 * ISBN을 받아 전국 도서관의 대출 가능 현황을 반환합니다.
 * </p>
 */
@Component
@RequiredArgsConstructor
@Slf4j
public class LoanCheckFunction {

    private final LibraryInfoNaruApiClient libraryInfoNaruApiClient;

    /**
     * 대출 가능 확인 실행 메서드 (외부에서 호출)
     */
    public LoanCheckResult checkLoanAvailability(LoanCheckRequest request) {
        log.info("[AI Function] 대출 가능 여부 조회 시작: isbn={}", request.isbn());

        try {
            // 1. 도서관정보나루 API 호출
            List<LoanItemInfo> loanItems = libraryInfoNaruApiClient.searchLoanItemsByIsbn(
                request.isbn()
            );

            if (loanItems.isEmpty()) {
                log.info("[AI Function] 대출 가능 도서관 없음: isbn={}", request.isbn());
                return LoanCheckResult.empty(request.isbn());
            }

            // 2. 지역 필터링 (선택 사항)
            List<LoanItemInfo> filteredItems = filterByRegion(loanItems, request.region());

            // 3. 거리 기반 정렬 (위도/경도 제공 시)
            if (request.latitude() != null && request.longitude() != null) {
                filteredItems = sortByDistance(
                    filteredItems,
                    request.latitude(),
                    request.longitude()
                );
            }

            // 4. 상위 N개 추출
            List<LoanItemInfo> topItems = filteredItems.stream()
                .limit(request.limit())
                .toList();

            // 5. 결과 변환
            List<LoanCheckResult.LibraryInfo> libraries = topItems.stream()
                .map(this::toLibraryInfo)
                .toList();

            log.info("[AI Function] 대출 가능 도서관 조회 완료: isbn={}, count={}",
                request.isbn(), libraries.size());

            return LoanCheckResult.of(
                request.isbn(),
                loanItems.size(),
                libraries
            );

        } catch (Exception e) {
            log.error("[AI Function] 대출 가능 여부 조회 실패: isbn={}", request.isbn(), e);
            return LoanCheckResult.empty(request.isbn());
        }
    }

    /**
     * 지역 기반 필터링
     *
     * @param loanItems 대출 도서 목록
     * @param region    지역
     * @return 필터링된 목록
     */
    private List<LoanItemInfo> filterByRegion(List<LoanItemInfo> loanItems, String region) {
        if (region == null || region.isBlank()) {
            return loanItems;
        }
        return loanItems.stream()
            .filter(item -> item.location() != null && item.location().contains(region))
            .collect(Collectors.toList());
    }

    /**
     * 거리 기반 정렬 (하버사인 공식)
     * <p>
     * 참고: 현재 LoanItemInfo에는 위도/경도 정보가 없으므로,
     * 지역 필터링만 수행하고 정렬은 생략합니다.
     * </p>
     */
    private List<LoanItemInfo> sortByDistance(List<LoanItemInfo> loanItems,
                                              Double latitude,
                                              Double longitude) {
        // TODO: 도서관 위도/경도 정보가 API 응답에 포함되면 거리 계산 구현
        log.debug("[AI Function] 거리 기반 정렬 요청되었으나, 위도/경도 정보 부족으로 생략");
        return loanItems;
    }

    /**
     * LoanItemInfo를 LibraryInfo로 변환합니다.
     */
    private LoanCheckResult.LibraryInfo toLibraryInfo(LoanItemInfo item) {
        boolean loanAvailable = "Y".equalsIgnoreCase(item.loanAvailable());

        return LoanCheckResult.LibraryInfo.of(
            item.libraryName(),
            item.location(),
            null, // 거리 정보 (현재 미지원)
            "",   // 전화번호 (현재 미지원)
            loanAvailable,
            ""    // 운영 시간 (현재 미지원)
        );
    }
}
```

## 4. AiLibraryAssistantService에 3개 Function 등록

### 4.1. AiLibraryAssistantService 최종 수정

**AiLibraryAssistantService.java** (`ai/service/AiLibraryAssistantService.java`):

이미 [03. 복수 Function 조합](./03.multiple-functions.md)에서 3개 Function을 모두 등록했습니다:

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class AiLibraryAssistantService {

    private final ChatModel chatModel;
    private final BookSearchFunction bookSearchFunction;
    private final ReviewFunction reviewFunction;
    private final LoanCheckFunction loanCheckFunction;

    public String ask(String userMessage) {
        log.info("[AI Service] 사용자 요청: {}", userMessage);

        // 3개의 FunctionToolCallback 생성
        FunctionToolCallback<BookSearchRequest, BookSearchResult> bookSearchTool =
            FunctionToolCallback.builder("BookSearch", bookSearchFunction::searchBooks)
                .description("도서관 시스템에서 도서를 검색합니다...")
                .inputType(BookSearchRequest.class)
                .build();

        FunctionToolCallback<Long, ReviewResult> reviewTool =
            FunctionToolCallback.builder("GetReviews", reviewFunction::getReviews)
                .description("도서의 리뷰 목록을 조회합니다...")
                .inputType(Long.class)
                .build();

        FunctionToolCallback<LoanCheckRequest, LoanCheckResult> loanCheckTool =
            FunctionToolCallback.builder("CheckLoanAvailability", loanCheckFunction::checkLoanAvailability)
                .description("도서의 대출 가능 여부를 조회합니다...")
                .inputType(LoanCheckRequest.class)
                .build();

        // ChatClient 생성 (3개 Function Tool 등록)
        ChatClient chatClient = ChatClient.builder(chatModel)
            .defaultToolCallbacks(bookSearchTool, reviewTool, loanCheckTool)
            .build();

        // System Prompt
        String systemPrompt = """
            당신은 도서관 도우미 AI입니다.

            복잡한 요청 처리 순서:
            1. 도서 검색이 필요하면 BookSearch 호출
            2. 리뷰가 필요하면 GetReviews 호출 (bookId 필요)
            3. 대출 가능 여부가 필요하면 CheckLoanAvailability 호출 (ISBN 필요)

            사용자에게 친절하고 상세한 답변을 제공하세요.
            """;

        String response = chatClient.prompt()
            .system(systemPrompt)
            .user(userMessage)
            .call()
            .content();

        return response;
    }
}
```

## 6. 에러 처리 및 Fallback

### 6.1. API 장애 시 Fallback

**시나리오**: 도서관정보나루 API 다운

**구현 방법**:
```java
try {
    var availability = libraryInfoNaruApiClient.getLoanAvailability(isbn);
} catch (Exception e) {
    log.error("[Function] API 호출 실패", e);

    // Fallback: 캐시된 데이터 확인
    var cached = cacheService.get("loan:" + isbn);
    if (cached != null) {
        return cached;
    }

    // 최후의 수단: 에러 메시지 반환
    return LoanCheckResult.builder()
        .isbn(isbn)
        .totalLibraries(0)
        .libraries(java.util.List.of())
        .build();
}
```
