# Step 8: n8n 워크플로우 자동화

## 학습 개요

Step 7에서 구현한 **Spring AI Function Calling 시스템**을 n8n과 연동하여 복잡한 워크플로우를 시각화하고 자동화합니다. 코드 없이 노드 기반으로 AI 시스템을 확장하고 운영하는 방법을 배웁니다.

## 학습 목표

1. **n8n 기초 이해**
   - 워크플로우 자동화 개념
   - 노드 기반 프로그래밍
   - Docker로 n8n 설치 및 실행

2. **Spring AI API와 n8n 연동**
   - HTTP Request 노드로 API 호출
   - Webhook 트리거로 외부 이벤트 수신
   - JSON 데이터 전송 및 수신

3. **실무 워크플로우 구축**
   - 도서 검색 → Slack 알림
   - 리뷰 요약 자동화
   - 신간 도서 자동 수집
   - 멀티 단계 오케스트레이션

4. **운영 및 모니터링**
   - 워크플로우 실행 내역 추적
   - 에러 핸들링
   - 성능 최적화

```
   [Slack 알림]
   [DB 로그 저장]
   [이메일 전송]
   [추가 처리]
```

## n8n이 추가되면 할 수 있는 것

### 1) 시각적 워크플로우 관리

**코드로 구현할 때**:
```java
// 함수 순서를 바꾸려면 코드를 다시 컴파일해야 함
searchBooks() → getReviews() → checkLoanAvailability()
```

**n8n으로 구현할 때**:
```
[검색] → [리뷰] → [대출확인]
   ↓
드래그 앤 드롭으로 순서 변경
바로 실행 가능
```

### 2) 외부 서비스 연동

- **Slack**: 검색 결과 알림
- **Email**: 리뷰 요약 전송
- **Google Sheets**: 검색 로그 저장
- **Telegram Bot**: 메시지 발신

### 3) 자동화 및 스케줄링

- **매일 오전 9시**: 신간 도서 자동 수집
- **리뷰 10개마다**: AI 요약 자동 생성
- **검색 100회마다**: 통계 리포트 생성
